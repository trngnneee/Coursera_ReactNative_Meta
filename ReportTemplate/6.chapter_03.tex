\section{Video: What is Tab Navigation}
\begin{itemize}
    \item Its a React Navigation library for setting up tab-based navigation between screens.
\end{itemize}


\subsection{Stack Navigation vs Tab Navigation}
\begin{itemize}
    \item Stack navigation can transition between screens and navigation history.
    \item With Tab navigation, you can contain tabs on the bottom of the screen or on the header. Clicking on a tab takes you a specific screen. $\rightarrow$ common design on modern mobile app.
\end{itemize}

\subsection{Options}
\begin{itemize}
    \item Bottom tab navigation
    \item Material design-based bottom tab navigation
    \item Material design-based top tab navigation
    \item Within the tab navigator screen options such as icons, active and inactive tint colors are readily available.
\end{itemize}
