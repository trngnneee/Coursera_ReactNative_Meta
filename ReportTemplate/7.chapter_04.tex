\section{Video: What is Drawer Navigation}
\subsection{Drawer Navigation}
\begin{itemize}
    \item The drawer appears when the user touches the hamburger icon in the app bar or when the user swipes a finger from the edge og the screen.
    \item Drawer navigation is, like Tab navigation, a common navigation pattern. It uses a drawer from left to right side for navigating between screens. 
    \item You would use this pattern typically when there are many screens to navigate to. 
\end{itemize}

\subsection{Syntax}
Import
\begin{lstlisting}[language=Java, numbers=none]
    import { createDrawerNavigator } from '@react-navigation/drawer'

    const Drawer = createDrawerNavigator();
\end{lstlisting}

Use in App
\begin{lstlisting}[language=Java, numbers=none]
    <Drawer.Navigator>
        <Drawer.Screen name='...' component={...} />
    </Drawer.Navigator>
\end{lstlisting}

\begin{itemize}
    \item We can add option to open the drawer on the right of the screen.
    \item We can also add multiple choices for the drawer, for example the navigation in the left drawer, and the contact in the right drawer,...
\end{itemize}
